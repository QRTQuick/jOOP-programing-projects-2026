Simple modular example demonstrating encapsulation and interaction between `Dog` and `Human` classes.

Structure:
- src/projectone/Dog.java
- src/projectone/Human.java
- src/projectone/Main.java

Compile and run (Windows PowerShell):

```powershell
cd "project one encapsulation oop for java"
javac -d out src\projectone\*.java
java -cp out projectone.Main
```

This will print object info and show method interactions (adopt, feed, walk, bark, sit).
